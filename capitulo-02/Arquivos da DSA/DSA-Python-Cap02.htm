Descreva o que é a Linguagem Python.

Python é uma linguagem de programação de alto nível, de propósito geral, interpretada e de código aberto. Ela foi criada por Guido van Rossum em 1989 e tem como objetivo a legibilidade do código e a capacidade de expressar conceitos em poucas linhas de código. Python é amplamente utilizado em aplicações de Ciência de Dados, Inteligência Artificial, Aprendizado de Máquina, Desenvolvimento Web, entre outras. Ele é compatível com vários sistemas operacionais, como Windows, MacOS e Linux.

Como surgiu a Linguagem Python?
1 / 2

A linguagem Python foi criada por Guido van Rossum nos anos 80, enquanto ele trabalhava na Universidade de Amsterdã. O nome "Python" foi escolhido como uma referência ao grupo de humor britânico Monty Python, de quem ele é fã.

A inspiração para criar a linguagem Python veio de outras linguagens de programação, como ABC, Modula-3, C, Shell Unix e outras. Guido visava criar uma linguagem mais simples e concisa, mas ao mesmo tempo poderosa e expressiva, que fosse fácil de ler e escrever. Ele também queria que a linguagem fosse livre e de código aberto, para que qualquer pessoa pudesse usá-la e contribuir para o seu desenvolvimento.

A primeira versão do Python foi lançada em 1991 e desde então, tem evoluído constantemente, com novas versões sendo lançadas regularmente. Hoje, a linguagem Python é amplamente utilizada em muitas áreas, desde ciência de dados e inteligência artificial até desenvolvimento web e automação de tarefas.

Gere código Python que crie uma lista com os números entre 1 e 100 e então imprima os números pares, mas somente se o número for divisível por 4.
# Criando uma lista com os números de 1 a 100
lista = list(range(1, 101))

# Usando um laço for para percorrer a lista
for num in lista:
    # Verificando se o número é par e divisível por 4
    if (num % 2 == 0) and (num % 4 == 0):
        print(num)

Este código cria uma lista chamada "lista" que contém os números de 1 a 100 usando a função range(). Em seguida, usa um laço for para percorrer a lista e verifica se o número atual é par e divisível por 4 usando o operador módulo (%), se for verdadeiro ele imprime o número na tela.

Esse código imprime todos os números pares que são divisíveis por 4 entre 1 e 100. Isso inclui os números 4, 8, 12, 16, etc.

Gere código de Python que crie uma lista com os números entre 1 e 100 e então imprima os números pares, mas somente se o número for divisível por 4, usando list comprehension 
# Criando uma lista com os números de 1 a 100
lista = [num for num in range(1, 101)]

# Usando list comprehension para filtrar e imprimir os números pares divisíveis por 4
pares_div4 = [num for num in lista if (num % 2 == 0) and (num % 4 == 0)]

print(pares_div4)

Este código cria uma lista chamada "lista" que contém os números de 1 a 100 usando a list comprehension, em seguida, é feita outra list comprehension para filtrar os números pares e divisíveis por 4 e armazenar em uma nova lista "pares_div4". Por fim, é impresso essa lista.

Esse código imprime todos os números pares que são divisíveis por 4 entre 1 e 100. Isso inclui os números 4, 8, 12, 16, etc.

List comprehension é uma maneira concisa e eficiente de criar e filtrar listas em Python, e pode ser mais fácil de ler e manter do que usar loops tradicionais.
